搭建博客


hexo写博客时怎么插入图片:

  • 以管理员模式运行git bash
  • 首先安装nodejs和npm,这在上一篇关于npm的问题博客已经安装,包括一些镜像源的问题都以更改
  • npm install -g hexo-cli 使用此命令安装hexo
  • hexo -v 查看是否安装好,可能显示无此命令,需要配置其环境变量
  • 成功后,如下图
  • cd /c/Users/pickg/blog创建一个blog目录并进入此目录
  • hexo init 在此初始化




部署到GitHub上

  • 登录GitHub并且新建一个仓库

  • 设置当前目录下的_config.yml文件
  • hexo d 推到github上
  • pickgog.github.io推上去就可以浏览器在输入这个就能进入博客

自定义一个主题

yilia主题

  • git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia 克隆下来


  • 在输入hexo d推到GitHub上
